# OverTheWire Bandit Solutions

This repository contains my personal solutions and command-line walkthrough for the Bandit challenges from OverTheWire.

Bandit is a beginner-friendly Linux wargame designed to teach the fundamentals of Linux, shell usage, and basic security concepts. Each level introduces a new challenge that requires exploring the system, understanding file permissions, using command-line tools, and applying problem-solving techniques to retrieve the password for the next level.

The goal of this repository is to document the commands, techniques, and reasoning used to solve each level from Bandit 0 to Bandit 32.
